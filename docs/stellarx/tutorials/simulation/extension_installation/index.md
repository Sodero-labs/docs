# StellarX Isaac Sim Extension Installation

You need a dedicated Isaac Sim extension to control robots with StellarX in the simulator.

## Extension Installation

![main menu](images/isaacsim_extension_00.png){ width="1000" }  

1. When you open Isaac Sim, you'll see the above default main screen.  
&nbsp;

![window](images/isaacsim_extension_01.png)  
![extensions](images/isaacsim_extension_02.png)  
2. Click the 'Window' tab and select the 'Extensions' menu.
&nbsp;

![extensions window](images/isaacsim_extension_03.png){ width="1000" }  
![extensions settings](images/isaacsim_extension_04.png){ width="1000" }  
3. On the extensions window, click the `THIRD PARTY` button. Delete default `feature`, and search for `stellar`.
&nbsp;

![extensions settings](images/isaacsim_extension_05.png){ width="1000" }  
4. Open up `Community` tab, and you'll see `STELLAR BRIDGE` extension. Toggle the button to enable it.
&nbsp;

![extensions window](images/isaacsim_extension_06.png){ width="1000" }  
5. Or you can click the extension and turn on `AUTOLOAD` to automatically enable stellar_bridge when you re-open Isaac Sim.
&nbsp;

And that's it! Your Isaac Sim is now prepared for StellarX. Follow [the next chapter](../robot_setup/index.md) to learn how to prepare a robot model.
&nbsp;
