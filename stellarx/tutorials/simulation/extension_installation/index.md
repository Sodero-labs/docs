# StellarX Isaac Sim Extension Installation

You need a dedicated Isaac Sim extension to control robots with StellarX in the simulator.

## Extension Installation

1. When you open Isaac Sim, you'll see the above default main screen.

1. Click the 'Window' tab and select the 'Extensions' menu.

1. On the extensions window, click the `THIRD PARTY` button. Delete default `feature`, and search for `stellar`.

1. Open up `Community` tab, and you'll see `STELLAR BRIDGE` extension. Toggle the button to enable it.

1. Or you can click the extension and turn on `AUTOLOAD` to automatically enable stellar_bridge when you re-open Isaac Sim.

And that's it! Your Isaac Sim is now prepared for StellarX. Follow [the next chapter](https://docs.sodero.io/stellarx/tutorials/simulation/robot_setup/index.md) to learn how to prepare a robot model.
